Kpop Starter Kit

  • Spotify Music Hackathon
  • September 2015
  • Project URL: http://kpopstarterkit.com/

The last time I updated the site was in 2018, so there are a few broken images and a few broken API calls.

I made this website with a couple of friends because we love kpop and wanted to make it accessible to everyone. We know that getting into kpop or learning about it can be intimidating because there are many groups and a language barrier, and we wanted the website to help people not only familirize themselves with to know 3 popular kpop groups' music, but also and its members, videos, and current activities.

We used a bootstrap template to make the website. In order to make the music accessible to people, we used the Spotify API to link playlists and albums, the YouTube API to showcase music videos and videos that show the members' personalities, and the Twitter API to let people know what is going on currently with each group.

I also ended up making the website into an iOS app during Fall 2017 at my internship using Objective-C.
