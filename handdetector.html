Hand Detector

  • Visual Interfaces to Computers
  • Professor John Kender
  • Spring 2020
  • Project URL: https://github.com/analeonnyc/handdetector

In this project, I had to create a program that read two hand gestures in order to lock a safe. The first gesture is a fist in the center of the image. The second image is an open hand, palm down, in any corner of the image.

In order to correctly identify the correct hand gestures to lock the safe, the program takes in a calibration skin patch of the hand in the image, gets the binarized image of the hand, cleans the image with a filter and blur, thresholds the image, uses the canny edge detector to outline the edge, and applies my hand detection algorithms.

I identified false positives and false negatives. I also created two new gestures that will lock the safe (two hand gesture).

In the link, you will find detailed documentation as well as the code.
