Using Machine Learning to Make a Map from a Distances Matrix

  • Machine Learning
  • Professor Nakul Verma
  • Summer 2020
  • Project URL: https://github.com/analeonnyc/distancestolatlong

In this assignment, I had to find the latitude and longitude coordinates from a matrix of relative distances between nine popular US citites.

In order to find the coordinates, I used a minimizer, an unconstrained optimization problem, and displayed the map.

My map ended up very accurate.
